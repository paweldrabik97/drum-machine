# Drum Machine

## Description

This project is a drum machine or sampler that plays a specific sound and displays its name when the corresponding button is pressed on the screen with the mouse or a key is pressed on the keyboard. This project was created as a project for the freeCodeCamp Front End Development Libraries certification.

## Features

*   Simulates a drum machine.
*   Plays sounds when a button is clicked or a key is pressed.
*   Displays the name of the sound in a designated display area.
*   Keyboard support for triggering sounds.
*   Uses React and Redux for state management.
*   Styled with Bootstrap.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
*   React
*   Redux
*   React-Redux
*   Bootstrap

## Installation

1.  Clone the repository:
    ```
    git clone https://github.com/paweldrabik97/drum-machine.git
    ```
2.  Open `index.html` in your browser.

## Usage

1.  Click on the drum pads on the screen or press the corresponding keys on your keyboard (Q, W, E, A, S, D, Z, X, C) to play the sounds.
2.  The name of the sound will be displayed in the display area.
